# Documentation Agent Notes

## Scope

Tài liệu là một phần của sản phẩm, không phải ghi chú phụ. Repo này dùng tiếng Việt có dấu cho tài liệu vận hành và học thuật, giữ tiếng Anh cho tên chuẩn/công nghệ khi cần.

## Rules

- Khi thêm chuẩn y tế, dẫn nguồn chính thức hoặc ghi rõ là giả định.
- Khi đổi Docker, CI, API base path hoặc command, cập nhật `README.md`, `AGENTS.md` và runbook liên quan.
- Không biến `docs/reports` hoặc `docs/research` thành source of truth runtime.
- Mermaid diagram phải dùng label rõ, tránh node mơ hồ.
